## 详细需求
- 协作系统: ERP人员的待办以及已办事项的管理系统
   1. 日历工作表系统: 存在三个权限流转层，公司层、部门层、个人层。公司层即为超管的日历表，部门层即为团队负责人的日历表，个人层即为普通员工的日历表。工作事项可以在三层中流转，工作事项的数据来源为ERP其余模块的数据
   2. 个人工作计划表: 每个用户都有个人工作计划表，用于记录自己的工作计划和任务。

- 产品管理系统
   1. 产品信息管理：包括公司多条业务线的产品的名称、描述、规格、价格、库存等信息
   2. 产品的历史信息包含更改人、更改时间、更改前的信息、更改后的信息等   
   
- 销售管理系统
   1. 线上销售，包括视频管理、图文推广管理、直播管理、官网管理的运营数据可以同步到销售管理系统并实时管理和数据面板分析
   2. 线下销售，包括代理商管理
   3. 所有渠道得到的客户需要推送到客户管理系统（CRM），生成基础的客户档案
   4. 平台对销售渠道进行数据统计和分析，并存储共享线上销售所产生的资料数据

- 客户管理系统（CRM）
   1. 客户状态：包括潜在客户、跟进中客户、已成交客户、未成交客户、待分配客户等状态
   2. 客户分配机制：实现客户的自动或手动分配给相应销售人员
   3. 客户信息管理：从销售管理系统中获取客户前期沟通数据形成客户档案
   4. 对客户的意向度和重要时间节点，例如客户生日、客户录入时间，成交时间等，推送对应通知到对应负责人协作日历工作表，提高客户转化度
   5. 支持批量导入客户和手动录入客户
   6. 完善客户档案中的客户沟通记录、客户交易记录、客户评价记录、客户家庭信息、考试场次信息等
   7. 对重复客户进行合并或者提醒处理，避免客户信息重复存储


- 订单和审批系统
  1. 由管理员和销售确认订单服务内容，确认合同后，生成合同。并提供给客户进行签约
  2. 确认订单，生成合同，签署合同等步骤节点都需要对应负责人进行审批
  3. 订单的收款、付款等资金操作数据推入财务系统
   
- 客户服务系统
  1. 教学教务服务、海外管家服务，面试服务，培训服务，申请服务。涉及到订单的审批流转和收款退款等数据操作

- 财务系统
  1. 资金池，对每一个公户的资金和流水进行管理，可以批量操作和手动操作，可以展示总资金和总流水
  2. 对业务订单的收款、付款等资金操作进行记录，生成业务财务流水
  3. 对员工的工资、奖金、扣款等操作进行记录，生成员工财务流水
  4. 对内部转账、采购、发票报销等操作进行记录，生成财务流水
  5. 生成期末财务报告，包括业务财务报告、员工财务报告、资金池报告等
  6. 对业务订单的应收、待收、退款、支出等进行记录和分析
  7. 对员工的提成和代理商的佣金发放进行结算和分析
  8. 对员工的薪资进行规则结算和分析
  9. 对大额收支、订单退赔、部门预算、个人报销进行风险管控提醒
  


- 职员管理系统 
  1. 职员编辑展示个人信息、合同、评估、考核、绩效、考勤
  2. 团队架构管理：团队架构图、团队架构说明、员工绩效分配，考勤规则分配等
  3. 考核管理系统：包括员工的绩效考核、提成考核、考勤考核
  4. 权限管理系统：公司权限分配、部门权限分配、每个员工的权限分配、代理商权限分配
   

- AI模块
  1. 销售培训模型： 销售团队可以通过AI模块进行销售培训，包括销售技巧、销售流程、产品介绍、产品分类、客户服务等。
  2. 客户服务模型： 客户可以通过AI模块进行填表提示、优化填表内容和格式。
  3. 文书优化模型： 销售团队可以通过AI模块对客户申请等文书进行优化，包括格式规范、内容完善优化等。
  4. 做题服务模型： 通过多个AI模型，对题目进行分析和解答，并交叉验证答案，解决模型幻读
   
- 系统底座
  1. 系统日志、系统监控
  2. 水印、隐私提示
  3. 智能部署管理
  4. 系统版本更新管理
  5. 企业微信接入
  6. 消息提示接入
  7. 系统基础UI样式更新
  8. 适配移动端布局（审批，客户服务，人事审批等等）
