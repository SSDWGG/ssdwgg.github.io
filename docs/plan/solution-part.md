> [!TIP]  教学教务服务
>- 教学教务1.0的需求已经上线，教师账号已经开通完成
>- 2.0需求如下：
>- 学生：存在一小时和45分钟的课时 统计：按课时
>- 老师：统计按小时（排课也需要有休息日） 
>- 需要有学生自己的排课（生成老师上课的时间，统计课时），需要设置休息日，学生还要额外排课（不计入课时统计）
>- 老师会上课到晚上11点，每周课表需要多一个备注功能，告知老师每日打卡时间
>- 学生信息：添加一个字段，考试规划情况（文本域）
>- 会有测试课表（不计入学生课时，也不计入老师课时，按次给老师结算）
>- 提供根据开始时间和结束时间查询老师上课的总课时统计
