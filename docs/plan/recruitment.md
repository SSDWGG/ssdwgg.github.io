# 技术部门人员招聘

## 招聘流程
> [!IMPORTANT]
> 1. 人事部门根据JD初筛简历
> 2. 技术部门给出的初筛评估，并给出合适的初筛面试题目
> 3. 人事部门根据初筛面试题目，对候选人进行二次筛选
> 4. 技术部门对候选人进行首轮面试，评估候选人的技术能力和项目经验
> 5. 项目负责人进行二轮面试，评估候选人的技术能力和团队合作精神
> 6. CTO进行最终评估，确定是否录用

## JD要求

### 中级前端开发工程师
1. 三年以上前端开发经验，具备复杂 Web 应用或 ERP 产品的实战经验。
2. 精通 HTML/CSS/JavaScript/TypeScript，熟练掌握 vue3、React18+ 前端框架，具备良好的工程化思维。
3. 熟悉 小程序/H5/桌面端/多端开发。
4. 熟悉前端构建工具链（如 Vite、Webpack）。
5. 在前端架构设计、性能调优、稳定性保障、单元测试等方面有实践经验。
6. 能熟练使用 AI 编程工具辅助开发，敢于尝试新技术与新交互范式。
7. 具备良好的沟通协作能力、产品意识和用户视角，能主动推动项目落地。
8. 有工作流、流程引擎、自定义表单、报表等相关经验者优先。

> ### 人事部门二次筛选要求
>1. 工作年限严格达到JD要求，且不存在不合理GAP期，且不能跳槽过于频繁 
>2. 必须熟练使用vue3 和 TypeScript 进行过一年以上的产品开发（询问是否熟练使用TS,简单讲讲对TS基本类型和Ts的工具函数，简单讲讲Vue3的响应式原理和对Hooks的理解）
>3. 必须了解或参与过至少一个小程序或者多端或者H5的项目开发（询问是否有了解过小程序或者多端开发）
>4. 必须了解前端性能优化相关知识，如代码分割、懒加载、缓存策略等（询问是否有过相关配置经验）




### 中级后端开发工程师
1. 三年以上开发经验， 具有中大型企业级应用系统后端开发经验，有OA、ERP、工作流引擎等相关项目经验者优先。
2. 精通Java，扎实的Java编程基础，熟悉常用设计模式、多线程、JVM，包括内存模型、类加载机制以及性能优化，具有系统设计、系统架构能力。
3. 熟悉主流开源框架，如 Spring Boot、Spring MVC、MyBatis 等主流开源框架，理解 Spring 核心原理。
4. 熟练掌握MySQL等关系型数据库，了解数据库基本原理，具备系统数据库设计能力，具备 SQL 优化经验，熟悉至少一种主流NoSQL数据库（如Redis、Elasticsearch等）。
5. 熟悉Linux操作系统，掌握常用的命令操作，能进行前后端服务的部署、监控和维护，了解Docker、Kubernetes等容器化技术者优先。
6. 拥有良好的学习能力，敢于尝试新技术，并能合理运用到项目中，附能为业务服务。
7. 工作认真负责，具备良好的团队协作意识执行力，逻辑清晰、思路有条理，具备良好的学习能力，能把复杂问题讲清楚、做扎实。


> ### 人事部门二次筛选要求
>1. 要求做过相关项目，例如OA、ERP、工作流引擎等。
>2. 熟悉Linux系统（询问在Linux系统中如何查询占用内存或CPU过高的进程？）
>3. 熟悉docker（询问docker中镜像与容器的关系）
